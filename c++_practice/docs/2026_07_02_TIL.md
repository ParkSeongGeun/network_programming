### Makefile
파일 간의 의존 관계 + 빌드 규칙을 미리 적어두는 파일
- make 명령어가 알아서 빌드해줌
    - 소스 안 바뀌면 재빌드 안함
    - Makefile 이름이어야함!

**[기본 문법]**
```
target: prerequisites
    recipe
```
- target: 결과물(만들고 싶은 파일)
- prerequisites: 필요한 파일들(의존성)
- recipe: 실제 실행할 명령어

**example**
```
main.o: main.c
    gcc -c main.c -o main.o
```

**[동작 원리]**
파일의 수정 시각을 비교해서 판단
- target이 prerequisites보다 최신 -> 스킵
- target이 없거나 prerequisites가 더 최근에 수정됨 -> 재실행

변수 활용도 가능
```
CC = gcc
CFLAGS = -Wall -O2

main.o: main.c
	$(CC) $(CFLAGS) -c main.c -o main.o
```
